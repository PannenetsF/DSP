\homep{2.11}

由于 LTI 不会改变信号的频率，分解

\[x(n) = \frac{1}{2 j} \left(e^{j \frac{\pi}{4} n} - e^{-j \frac{\pi}{4} n}\right)\]

那么 

\[y(n) = \frac{1}{2 j} \left(H(e^{j \pi/4})e^{j \pi n / 4  } - H(e^{-j \pi/4}) e^{-j \pi n / 4} \right)\]


带入公式

\[H(e^{j \pi / 4}) = \frac{1 - (- j)}{1 - 1/2} = 2\sqrt{2}e^{j \pi/4}\]     

\[H(e^{-j \pi / 4}) = \frac{1 - j}{1 - 1/2} = 2\sqrt{2}e^{-j \pi/4}\]     

代回

\[y(n) = 2 \sqrt{2} \sin (\frac{\pi}{4} (n+1))\]
